## 🎯 Как использовать админку:

### 1. **Отправка выбранным пользователям:**

1. Перейдите в админку Django: `/admin/teachers/telegramuser/`
2. Выберите пользователей, которым хотите отправить сообщение (поставьте галочки)
3. В выпадающем меню "Действие" выберите:
   - **📤 Отправить сообщение выбранным пользователям** - для Telegram пользователей
   - **👤 Отправить сообщение Django пользователям** - для Django аккаунтов
4. Нажмите "Выполнить"
5. Введите текст сообщения
6. Нажмите "Отправить сообщение"

### 2. **Массовая рассылка всем:**

1. На странице Telegram пользователей нажмите кнопку **📢 Массовая рассылка**
2. Выберите тип получателей:
   - **Всем пользователям** - все кто нажал /start
   - **Только учителям** - только учителя
   - **Только ученикам** - только ученики
3. Введите текст сообщения
4. Нажмите "Отправить рассылку"

Пользователь должен:

## 🚨 Если не работает:

### **Проверьте:**
1. **Бот запущен:** `./start_bot.sh`
2. **Пользователи нажали /start** в боте
3. **У пользователей включены уведомления**
4. **Токен бота корректный** в настройках

### **Диагностика:**
```bash
# Проверить статус пользователей
pipenv run python manage.py telegram_users --action list

# Тестировать отправку
pipenv run python manage.py telegram_users --action test --username mirafzal

# Массовая рассылка через командную строку
pipenv run python manage.py telegram_users --action broadcast --message "Тест" --user-type all
```
