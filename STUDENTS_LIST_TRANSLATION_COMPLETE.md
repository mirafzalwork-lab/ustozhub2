## 🔧 Технические детали

### Добавленные теги Django i18n

1. **{% load i18n %}** - в начале файла
2. **{% trans 'текст' %}** - для статических строк
3. **_("текст")** - для строк в Python-контексте (фильтр default)

### Примеры использования

#### Заголовки
```django
<h1 class="page-title">{% trans 'Ученики ищут учителей' %}</h1>
```

#### Вкладки
```django
<span>{% trans 'Ученики' %}</span>
```

#### Фильтры
```django
<label class="filter-label">
    <i class="fas fa-search"></i> {% trans 'Поиск' %}
</label>
```

#### Плейсхолдеры
```django
placeholder="{% trans 'Имя ученика...' %}"
```

#### Множественные формы
```django
{% if students.count == 1 %}{% trans 'ученик' %}
{% elif students.count < 5 %}{% trans 'ученика' %}
{% else %}{% trans 'учеников' %}{% endif %}
```

#### Фильтр default
```django
{{ student.description|default:_("Ищу учителя для изучения новых предметов") }}
```
