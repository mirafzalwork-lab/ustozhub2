Как это работает: утренняя и вечерняя рассылка запускается cron-скриптом: python manage.py send_daily_reminder --period morning и --period evening. Для каждого пользователя берётся активный шаблон его языка (ru / uz / en), если вариантов несколько — выбирается случайно. Если для языка нет активных шаблонов — используется русский. Поддерживается Markdown: *жирный*, _курсив_, эмодзи.
