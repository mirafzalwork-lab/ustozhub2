## 🎯 Как использовать

### **Через админ панель:**

1. **Индивидуальная отправка:**
   - Перейдите в `Telegram пользователи`
   - Выберите пользователей
   - Выберите действие `Отправить сообщение выбранным`
   - Введите текст и отправьте

2. **Отправка Django пользователям:**
   - Выберите действие `Отправить сообщение Django пользователям`
   - Система найдет связанные Telegram аккаунты

3. **Массовая рассылка:**
   - Нажмите кнопку `📢 Массовая рассылка`
   - Выберите тип получателей
   - Введите сообщение и отправьте

### **Через командную строку:**

```bash
# Посмотреть всех пользователей
pipenv run python manage.py telegram_users --action list

# Привязать пользователя
pipenv run python manage.py telegram_users --action link --telegram-id 7979988856 --username mirafzal

# Тестировать отправку
pipenv run python manage.py telegram_users --action test --username mirafzal
pipenv run python manage.py telegram_users --action test --telegram-id 7979988856

# Массовая рассылка
pipenv run python manage.py telegram_users --action broadcast --message "Важное объявление!" --user-type all
pipenv run python manage.py telegram_users --action broadcast --message "Только учителям" --user-type teacher
pipenv run python manage.py telegram_users --action broadcast --message "Только ученикам" --user-type student

# Отвязать пользователя
pipenv run python manage.py telegram_users --action unlink --telegram-id 7979988856
```
