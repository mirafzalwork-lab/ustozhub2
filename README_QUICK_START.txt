1️⃣ УСТАНОВКА ЗАВИСИМОСТЕЙ:

   pip install -r requirements.txt

2️⃣ ИНИЦИАЛИЗАЦИЯ БД:

   python manage.py migrate

═════════════════════════════════════════════════════════════════

3️⃣ СОЗДАНИЕ АДМИНИСТРАТОРА:

   python manage.py createsuperuser

═════════════════════════════════════════════════════════════════

4️⃣ ЗАПУСК СЕРВЕРА:

   python manage.py runserver

═════════════════════════════════════════════════════════════════

5️⃣ ТЕСТИРОВАНИЕ:

   Откройте в браузере:
   
   http://127.0.0.1:8000/en/register/step1/
   
   Заполните форму и дойдите до Шаг 2 (выбор предметов)
   
   ✅ Выберите предмет из поиска, введите цену, нажмите "Далее"
